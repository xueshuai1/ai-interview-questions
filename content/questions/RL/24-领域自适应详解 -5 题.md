# 领域自适应详解面试题（5 道）

## 题目 1：核心概念
**问题：** 什么是领域自适应？它与迁移学习有什么关系？

**参考答案：**
**领域自适应（Domain Adaptation）**：
迁移学习的子问题，将源域学到的模型适应到目标域，解决域间分布差异。

**与迁移学习的关系**：
```
迁移学习（广义）
├── 领域自适应（域不同，任务相同）
├── 任务迁移（任务不同）
└── 其他迁移形式
```

**核心问题**：
```
P_source(X,Y) ≠ P_target(X,Y)
```

**域差异类型**：

| 类型 | 公式 | 含义 |
|------|------|------|
| 协变量偏移 | P_s(X)≠P_t(X) | 输入分布不同 |
| 概念偏移 | P_s(Y\|X)≠P_t(Y\|X) | 条件分布不同 |
| 先验偏移 | P_s(Y)≠P_t(Y) | 标签分布不同 |

**自适应场景**：

**监督自适应**：
- 目标域有少量标签
- 最容易

**无监督自适应**：
- 目标域无标签
- 最常见

**半监督自适应**：
- 目标域部分标签
- 折中方案

**应用示例**：
```
仿真→真实：域不同（外观、物理）
室内→室外：域不同（光照、背景）
白天→夜晚：域不同（光照条件）
```

**评分要点：** 说出分布差异得基本分，解释类型得高分

## 题目 2：DANN 算法
**问题：** DANN 的核心思想是什么？梯度反转层如何工作？

**参考答案：**
**DANN（Domain-Adversarial Neural Network）**：
用对抗训练学习域不变特征。

**核心思想**：
```
特征提取器：提取特征
任务分类器：预测标签
域判别器：区分源/目标域

目标：
- 任务分类准确
- 域判别器无法区分（特征域不变）
```

**损失函数**：
```
L = L_task - λ L_domain
```
- L_task：任务损失（源域）
- L_domain：域分类损失
- λ：平衡系数

**梯度反转层（Gradient Reversal Layer）**：

**前向传播**：
```
GRL(x) = x  （恒等）
```

**反向传播**：
```
∂L/∂x = -λ · ∂L/∂GRL(x)
```
梯度乘以负系数，反转方向。

**效果**：
```
域判别器：最小化域分类损失
特征提取器：最大化域分类损失（通过 GRL）
→ 特征提取器学习欺骗判别器
→ 域不变特征
```

**实现**：
```python
class GradientReversal(torch.autograd.Function):
    @staticmethod
    def backward(ctx, grad_output):
        return -ctx.lambda_ * grad_output
```

**训练动态**：
```
初期：λ小，重点任务
后期：λ大，重视自适应
```

**优势**：
- 端到端训练
- 无需目标域标签
- 简单有效

**评分要点：** 说出对抗训练得基本分，解释梯度反转得高分

## 题目 3：MMD 对齐
**问题：** MMD 如何度量域间差异？如何用于领域自适应？

**参考答案：**
**MMD（Maximum Mean Discrepancy）**：
度量两个分布差异的非参数方法。

**定义**：
```
MMD²(P,Q) = ||E_P[φ(X)] - E_Q[φ(Y)]||²
```
其中φ是特征映射（如 RBF 核）。

**核技巧计算**：
```
MMD² = E[K(X,X')] + E[K(Y,Y')] - 2E[K(X,Y)]
```

**多核 MMD**：
```
MK-MMD = Σ β_k MMD²_k
```
多个核尺度，更鲁棒。

**用于领域自适应**：

**1. 特征对齐损失**：
```
L_total = L_task + λ · MMD²(F_s, F_t)
```

**2. 优化目标**：
```
min_θ L_task + λ · MMD²(φ_θ(X_s), φ_θ(X_t))
```

**3. 效果**：
- 源域和目标域特征分布接近
- 分类器在目标域也有效

**实现**：
```python
def mmd_loss(source, target, sigma=1.0):
    # RBF 核
    K_ss = rbf_kernel(source, source, sigma)
    K_tt = rbf_kernel(target, target, sigma)
    K_st = rbf_kernel(source, target, sigma)
    
    return K_ss.mean() + K_tt.mean() - 2 * K_st.mean()
```

**与对抗方法对比**：

| 方面 | MMD | 对抗 |
|------|-----|------|
| **稳定性** | 高 | 中 |
| **计算** | 简单 | 复杂 |
| **效果** | 好 | 更好 |
| **调参** | 核参数 | 多个超参 |

**实践建议**：
- 多核 MMD 更鲁棒
- 结合对抗方法
- 监控 MMD 值

**评分要点：** 写出 MMD 公式得基本分，对比对抗方法得高分

## 题目 4：自训练方法
**问题：** 自训练如何实现领域自适应？伪标签有什么风险？

**参考答案：**
**自训练（Self-Training）**：
用模型自己生成的伪标签训练目标域。

**算法流程**：
```
1. 源域训练模型 M
2. M 预测目标域样本→伪标签
3. 选择高置信度样本
4. 用伪标签训练 M
5. 重复 2-4
```

**伪标签生成**：
```python
with torch.no_grad():
    preds = model(target_data)
    probs = softmax(preds)
    confidences, pseudo_labels = max(probs, dim=1)

# 选择高置信度
mask = confidences >= threshold
```

**风险**：

**1. 错误累积**：
```
错误伪标签→错误训练→更差模型→更多错误
```

**2. 确认偏差**：
```
模型倾向于确认已有预测
忽略反例
```

**3. 类别不平衡**：
```
某些类别伪标签多
训练偏向这些类别
```

**缓解方法**：

**1. 高阈值**：
```
threshold = 0.9  # 只选高置信度
```

**2. 课程学习**：
```
初期：高阈值
后期：降低阈值
```

**3. 类别平衡**：
```
每类选择相同数量
避免偏差
```

**4. 一致性正则**：
```
增强样本→预测应一致
减少错误传播
```

**5. 教师 - 学生**：
```
教师模型生成伪标签
学生模型学习
教师 EMA 更新
```

**实践建议**：
- 从高阈值开始
- 监控伪标签质量
- 结合其他方法

**评分要点：** 说出伪标签风险得基本分，提出缓解方法得高分

## 题目 5：实际应用
**问题：** 领域自适应在实际应用中有什么挑战？如何选择合适的方法？

**参考答案：**
**实际应用挑战**：

**1. 无目标域标签**：
- 无法验证自适应效果
- 只能间接评估

**2. 域差异大**：
- 简单方法不够
- 需要复杂方法

**3. 计算资源**：
- 对抗训练成本高
- 部署受限

**4. 多域场景**：
- 多个目标域
- 需要通用模型

**5. 在线适应**：
- 流式数据
- 实时更新需求

**方法选择决策框架**：

**1. 评估域差异**：
```
差异小 → 简单方法（MMD）
差异大 → 复杂方法（对抗）
```

**2. 评估标签情况**：
```
有目标标签 → 监督自适应
无目标标签 → 无监督自适应
少量标签 → 半监督
```

**3. 评估资源**：
```
资源充足 → 对抗训练
资源有限 → MMD/自训练
```

**4. 评估实时性**：
```
在线需求 → 轻量方法
离线可接受 → 复杂方法
```

**推荐方案**：

| 场景 | 推荐方法 |
|------|----------|
| 小差异，无标签 | MMD 对齐 |
| 大差异，无标签 | DANN |
| 少量标签 | 半监督自训练 |
| 多目标域 | 多域自适应 |
| 在线适应 | 轻量自训练 |

**实践流程**：
```
1. 分析域差异
2. 选择基线方法
3. 评估效果
4. 迭代改进
5. 监控部署性能
```

**成功案例**：
-  Office-31（图像分类）
-  VisDA（仿真→真实）
-  跨语言 NLP

**评分要点：** 说出 3 个以上挑战得基本分，给出决策框架得高分
