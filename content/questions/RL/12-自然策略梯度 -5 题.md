# 自然策略梯度面试题（5 道）

## 题目 1：核心概念
**问题：** 什么是自然策略梯度？它与普通策略梯度有什么区别？

**参考答案：**
**自然策略梯度（NPG）**：
在策略分布空间而非参数空间中进行最速下降，使用 Fisher 信息矩阵对梯度进行预处理。

**公式**：
```
普通梯度：∇_θ J(θ)
自然梯度：F(θ)^{-1} ∇_θ J(θ)
```
其中 F 是 Fisher 信息矩阵。

**区别**：

| 方面 | 普通梯度 | 自然梯度 |
|------|----------|----------|
| 空间 | 参数空间 | 分布空间 |
| 度量 | 欧氏距离 | KL 散度 |
| 更新 | θ ← θ + α∇J | θ ← θ + αF^{-1}∇J |
| 几何 | 忽略曲率 | 考虑曲率 |
| 步长 | 固定 | 自适应 |

**直观理解**：
- 普通梯度：参数变化相同，但分布变化可能不同
- 自然梯度：保证分布变化一致
- 考虑了参数空间的"拉伸"和"压缩"

**示例**：
```
高斯 N(μ, σ²)
参数变化Δμ=1：
- σ=0.1：分布变化大
- σ=10：分布变化小

自然梯度自动调整
```

**评分要点：** 说出 Fisher 矩阵得基本分，解释几何意义得高分

## 题目 2：Fisher 信息矩阵
**问题：** 什么是 Fisher 信息矩阵？它在 NPG 中起什么作用？

**参考答案：**
**Fisher 信息矩阵定义**：
```
F(θ) = E_x∼p(x|θ)[∇_θ log p(x|θ) ∇_θ log p(x|θ)^T]
```

**性质**：
- 半正定对称矩阵
- 度量参数空间的局部曲率
- KL 散度的二阶近似

**与 KL 散度的关系**：
```
KL(θ || θ+Δθ) ≈ (1/2) Δθ^T F(θ) Δθ
```

**在 NPG 中的作用**：

**1. 度量分布变化**：
- F 大的方向：参数小变化→分布大变化
- F 小的方向：参数大变化→分布小变化

**2. 预处理梯度**：
```
自然梯度 = F^{-1} · 普通梯度
```
- F 大的方向：缩小梯度
- F 小的方向：放大梯度

**3. 约束优化**：
```
max_θ ∇J^T Δθ
s.t. Δθ^T F Δθ ≤ δ（KL 约束）
```
解为：Δθ ∝ F^{-1} ∇J

**4. 自适应步长**：
- 自动调整不同方向的更新幅度
- 避免过大或过小的策略变化

**评分要点：** 写出定义得基本分，解释 KL 关系得高分

## 题目 3：共轭梯度法
**问题：** 为什么 NPG 要用共轭梯度法？如何实现？

**参考答案：**
**为什么用共轭梯度法**：

**问题**：直接计算 F^{-1} 成本高
- F 是 d×d 矩阵（d=参数数量）
- 求逆复杂度 O(d³)
- 大规模网络不可行

**解决方案**：共轭梯度法
- 只需计算 F·v（矩阵向量乘积）
- 无需显式构造 F
- 复杂度 O(k·d)，k 是迭代次数

**共轭梯度算法**：

```
求解：F · x = b

初始化：x_0 = 0, r_0 = b, p_0 = r_0

迭代 k=0,1,...:
    α_k = (r_k^T r_k) / (p_k^T F p_k)
    x_{k+1} = x_k + α_k p_k
    r_{k+1} = r_k - α_k F p_k
    β_k = (r_{k+1}^T r_{k+1}) / (r_k^T r_k)
    p_{k+1} = r_{k+1} + β_k p_k
```

**F·v 的计算**（自动微分）：
```python
# g = ∇logp
g = grad(log_probs)

# F·v = ∇(g^T · v)
Fv = grad((g * v).sum())
```

**实现要点**：
1. 用自动微分计算 F·v
2. 通常 10-20 次迭代足够
3. 添加阻尼提高稳定性

**评分要点：** 说出避免求逆得基本分，写出算法得高分

## 题目 4：约束优化视角
**问题：** 从约束优化角度解释自然策略梯度。

**参考答案：**
**约束优化问题**：
```
max_Δθ ∇_θ J(θ)^T Δθ
s.t. KL(π_θ || π_{θ+Δθ}) ≤ δ
```

**含义**：
- 目标：最大化目标函数增长
- 约束：策略变化不超过δ（KL 散度）
- 避免过大更新导致性能下降

**求解**：

**1. KL 二阶近似**：
```
KL(π_θ || π_{θ+Δθ}) ≈ (1/2) Δθ^T F(θ) Δθ
```

**2. Lagrange 函数**：
```
L(Δθ, λ) = ∇J^T Δθ - λ((1/2)Δθ^T F Δθ - δ)
```

**3. 最优条件**：
```
∂L/∂Δθ = ∇J - λ F Δθ = 0
→ Δθ = (1/λ) F^{-1} ∇J
```

**4. 确定λ**：
代入约束：
```
(1/2)((1/λ)F^{-1}∇J)^T F ((1/λ)F^{-1}∇J) = δ
→ (1/λ)² (1/2) ∇J^T F^{-1} ∇J = δ
→ 1/λ = √(2δ / (∇J^T F^{-1} ∇J))
```

**5. 最终解**：
```
Δθ = √(2δ / (∇J^T F^{-1} ∇J)) · F^{-1} ∇J
```

忽略常数因子，得到自然梯度更新：
```
Δθ ∝ F^{-1} ∇J
```

**直观理解**：
- 在 KL 球内找最陡方向
- 自然梯度是最优解
- δ控制更新幅度

**评分要点：** 写出约束问题得基本分，推导最优解得高分

## 题目 5：与 TRPO 关系
**问题：** 自然策略梯度与 TRPO 有什么关系？TRPO 做了哪些改进？

**参考答案：**
**关系**：
- NPG 是 TRPO 的理论基础
- TRPO 是 NPG 的实用化改进
- 两者都基于自然梯度思想

**NPG 的局限**：

**1. 线搜索缺失**：
- NPG 用固定学习率
- 可能更新过大或过小

**2. 近似误差**：
- 共轭梯度是近似解
- 累积误差可能影响

**3. 实现复杂**：
- 需要 Fisher 向量乘积
- 调参困难

**TRPO 的改进**：

**1. 线搜索**：
```
θ_new = θ + α^k · Δθ_NPG
```
- 回溯线搜索保证改进
- 自动调整步长

**2. 精确 KL 约束**：
- 不用二阶近似
- 直接计算 KL 散度
- 更准确的约束

**3. 共轭梯度优化**：
- 更高效实现
- 更好的数值稳定性

**4. 优势估计**：
- 用 GAE 估计优势
- 减少方差

**对比**：

| 方面 | NPG | TRPO |
|------|-----|------|
| 步长 | 固定学习率 | 线搜索 |
| KL 约束 | 二阶近似 | 精确计算 |
| 理论保证 | 局部 | 单调改进 |
| 实现难度 | 中 | 高 |
| 实践效果 | 好 | 更好 |

**总结**：
- NPG：理论优雅
- TRPO：实践有效
- PPO：TRPO 简化版

**评分要点：** 说出 NPG 是基础得基本分，详细对比得高分
