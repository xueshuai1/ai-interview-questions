## 题目描述
大语言模型（Large Language Model, LLM）是 AI 领域的核心技术。请解释 LLM 的定义、训练方法、核心能力及应用场景。

**标签：** 大语言模型，预训练，微调，上下文学习，涌现能力

## 参考答案
### 定义

**大语言模型（LLM）** 是基于 Transformer 架构、在海量文本数据上预训练的大规模神经网络模型，具有强大的语言理解和生成能力。

**核心特征：** 大规模参数量（7B-万亿级）、海量训练数据、通用能力。

### 训练方法

**1. 预训练（Pre-training）**
- 目标：学习语言的通用表示
- 方法：自监督学习（预测下一个 token）
- 数据：互联网文本（网页、书籍、代码、对话等）

**2. 有监督微调（SFT）**
- 目标：让模型遵循指令
- 数据：人工标注的指令 - 回答对

**3. 对齐优化（RLHF/DPO）**
- RLHF：基于人类反馈的强化学习
- DPO：直接偏好优化，更简单高效

### 核心能力

**1. 语言理解：** 分类、问答、抽取
**2. 语言生成：** 创作、对话、翻译
**3. 推理能力：** 数学、逻辑、代码
**4. 上下文学习：** 通过示例学习任务模式
**5. 涌现能力：** 规模达到阈值后突然出现的能力

### 实际应用场景

| 场景 | 应用示例 |
|------|---------|
| 内容创作 | 文章写作、营销文案 |
| 编程辅助 | 代码生成、调试 |
| 客户服务 | 智能客服、问答机器人 |
| 知识问答 | 百科问答、专业咨询 |
| 教育辅导 | 题目讲解、语言学习 |

### 能力边界

| 局限 | 说明 |
|------|------|
| 知识截止 | 训练数据有时间限制 |
| 幻觉问题 | 可能生成错误内容 |
| 推理错误 | 复杂推理可能出错 |

## 延伸追问

### 追问 1: Tokenization 原理（5分）
**问题：** LLM 如何处理输入文本？

**答案：** 使用 BPE、WordPiece 等方法将文本切分为 token。平衡词表大小与覆盖率。

### 追问 2: 上下文窗口扩展（5分）
**问题：** 如何扩展 LLM 的上下文窗口？

**答案：** 稀疏注意力、线性注意力、滑动窗口、RoPE 外推、ALiBi。当前水平：128K-1M+ tokens。

### 追问 3: LLM 评估方法（5分）
**问题：** 如何评估 LLM 的性能？

**答案：** 基准测试（MMLU、GSM8K、HumanEval）、人工评估、A/B 测试。

### 追问 4: LLM 发展趋势（5分）
**问题：** LLM 技术未来有哪些发展趋势？

**答案：** 多模态融合、Agent 化、小型化、专业化、长上下文、推理优化。

## 深入理解
LLM 是 AI 领域的通用技术，正在重塑人机交互方式。理解 LLM 的基本原理、能力和局限，是有效使用和改进 LLM 应用的基础。
