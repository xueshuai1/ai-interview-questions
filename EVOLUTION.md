## 🔄 自我进化框架

### 核心机制：进化循环

```
执行前：
  读 EVOLUTION.md → 了解当前策略和待解决问题

执行中：
  按 AGENT-MISSION.md 执行任务

执行后（必须！）：
  1. 自我反思：
     - 做了什么？效果如何？
     - 同样的问题是否重复出现？
     - 发现了什么新机会？
     - 下次应该改变什么？
  2. 更新 EVOLUTION.md
  3. 如有策略调整 → 更新 AGENT-MISSION.md / HEARTBEAT.md / CONTENT-STRATEGY.md
  4. 提交推送（包含策略变更）
```

### 进化评分（每次自评 1-10）

| 分数 | 标准 |
|------|------|
| 10 | 发现全新渠道 + 更新策略 + 高质量内容 |
| 8-9 | 主动改变策略 + 发现新机会 + 有反思 |
| 6-7 | 执行正常 + 有反思 + 小改进 |
| 4-5 | 执行正常但无反思 |
| 2-3 | 重复同样的事，无改进 |
| 1 | 执行失败 + 无反思 |

**目标：平均分 ≥ 7**
