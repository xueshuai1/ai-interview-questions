## 注意事项

1. **数据源优先级**：网站优先从 `data/` 目录加载 JSON 文件，`content/` 和 `questions/` 目录的 MD 文件为向后兼容
2. **标题转换规则**：
   - 包含"机制"、"架构" → 添加"的原理是什么？"
   - 包含"对比"、"演进" → 添加"的区别是什么？"
   - 包含"方法"、"技巧" → 添加"有哪些？"
   - 其他 → 添加"是什么？"或"什么是...？"
3. **知识库保持文章形式**：不转换为问题形式，保持"详解"、"原理"等文章标题
