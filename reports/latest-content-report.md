## 🔥 TOP 5 热点

### 1. Meerkat：AI Agent 安全审计新框架，发现开发者作弊和奖励黑客攻击
- **来源**: arXiv (2604.11806, Apr 13)
- **时间**: 2026-04-13
- **摘要**: 研究者推出 Meerkat，结合聚类与 Agent 搜索来发现用自然语言描述的安全违规。无需种子场景、固定工作流或穷举枚举，即可发现稀疏失败案例。在滥用、错位和任务游戏设置中，Meerkat 显著优于基线监控器，**发现了顶级 Agent 基准测试中普遍存在的开发者作弊行为**，并在 CyBench 上发现了比之前审计多近 4 倍的奖励黑客攻击案例。
- **建议**: 研究/安全头条。AI Agent 安全是行业核心痛点，"发现开发者作弊"这个点极具传播性。

### 2. SWE-AGILE：软件 Agent 动态推理上下文框架，7B 小模型刷新 SWE-Bench 纪录
- **来源**: arXiv (2604.11716, Apr 13)
- **时间**: 2026-04-13
- **摘要**: 针对 ReAct 式自主软件工程缺乏 System-2 推理的问题，提出 SWE-AGILE 框架。引入"动态推理上下文"策略：维护即时推理的"滑动窗口"保持连续性，同时将历史推理压缩为简洁的"推理摘要"。仅用 2.2k 轨迹和 896 个任务，就在 SWE-Bench-Verified 上为 7B-8B 模型树立了新标准。
- **建议**: 开发者工具/研究。小模型在复杂工程任务上刷新纪录，对资源有限的开发者意义重大。

### 3. OpenAI 收购 TBPN，加速安全研究布局
- **来源**: OpenAI 官方博客 (Apr 2)
- **时间**: 2026-04-02
- **摘要**: OpenAI 宣布收购 TBPN（The Big Picture Network），这是一家专注于 AI 安全和可解释性研究的公司。结合 OpenAI 最近的 $1220 亿融资和 IPO 计划，这次收购表明 OpenAI 正大力强化安全研究能力。
- **建议**: 企业/安全新闻。OpenAI 在 IPO 前夕的战略收购值得关注。

### 4. OIDA：组织 AI 需要认知基础设施，而非仅仅是检索
- **来源**: arXiv (2604.11759, Apr 13)
- **时间**: 2026-04-13
- **摘要**: 提出 OIDA 框架，将组织知识结构化为带有认知类别、重要性分数和矛盾边的知识对象。引入 QUESTION 机制——将"组织不知道什么"作为一等公民建模，随时间增加紧迫性。在受控比较中，OIDA 的 RAG 条件（3,868 token）达到 EQS 0.530，而全上下文基线（108,687 token）为 0.848，token 预算差异 28.1 倍。
- **建议**: 研究/企业。对 RAG 和企业知识管理有深刻见解，适合深度博客。

### 5. Codex 推出按需付费团队定价
- **来源**: OpenAI 官方博客 (Apr 2)
- **时间**: 2026-04-02
- **摘要**: OpenAI 为 Codex 推出灵活的按需付费定价方案，面向团队用户。此前 Codex 主要采用订阅制，新定价让小型团队和按需使用的开发者更容易接入。
- **建议**: 产品/开发者。Codex 定价策略变化直接影响开发者采用意愿。
