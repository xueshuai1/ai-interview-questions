## 修复概述

修复了前端页面只渲染 `question.content` 的问题，实现了对结构化数据格式的完整支持。

## 修复内容

### 1. 题目详情页 (`src/app/questions/[id]/page.tsx`)

✅ **已实现功能：**

- **答案摘要渲染** - 显示 `answer.summary`，使用蓝色背景高亮
- **章节遍历渲染** - 遍历 `answer.sections` 数组，每个章节显示：
  - 章节编号和标题
  - 章节内容（支持 Markdown）
  - 代码示例（如果有 `section.code`）
- **考察重点三维度** - 渲染 `evaluation` 对象：
  - 📚 知识理解
  - 💪 能力要求
  - 🧠 思维能力
- **延伸追问** - 渲染 `followUpQuestions` 数组：
  - 显示总分数（默认 25 分）
  - 每题显示题号和分数
  - 支持可折叠的答案提示

✅ **向后兼容：** 如果题目没有 `answer` 字段，回退到渲染 `content`（旧格式）

### 2. 知识库详情页 (`src/app/knowledge/[category]/[articleId]/page.tsx`)

✅ **已实现功能：**

- **文章摘要** - 渲染 `abstract` 字段
- **章节结构** - 渲染 `sections` 数组：
  - 主章节标题和内容
  - 子章节（`subsections`）嵌套渲染
  - 支持 Markdown 和表格
- **关键要点** - 渲染 `keyTakeaways` 数组（绿色背景）
- **前置知识** - 渲染 `prerequisites` 数组（琥珀色背景）
- **代码示例** - 渲染 `codeExamples` 数组：
  - 代码语言标识
  - 代码高亮显示
  - 代码解释说明
- **图表** - 渲染 `diagrams` 数组
- **参考资料** - 渲染 `references` 数组，带链接和类型标签

✅ **向后兼容：** 如果文章没有 `sections` 字段，回退到渲染 `content`（旧格式）

### 3. API 更新 (`src/app/api/knowledge/[category]/route.ts`)

✅ **新增功能：**

- 支持 `articleId` 查询参数
- 返回单篇文章完整数据
- 返回相邻文章用于导航（prev/next）

## 技术细节

### 数据结构

**题目格式：**
```json
{
  "answer": {
    "summary": "...",
    "sections": [
      {
        "title": "...",
        "content": "...",
        "code": {
          "language": "python",
          "content": "...",
          "explanation": "..."
        }
      }
    ]
  },
  "evaluation": {
    "knowledge": "...",
    "ability": "...",
    "thinking": "..."
  },
  "followUpQuestions": [
    {
      "question": "...",
      "points": 5,
      "answerHint": "..."
    }
  ]
}
```

**知识库文章格式：**
```json
{
  "abstract": "...",
  "sections": [
    {
      "id": "sec-1",
      "title": "...",
      "level": 1,
      "content": "...",
      "subsections": [...]
    }
  ],
  "keyTakeaways": ["..."],
  "prerequisites": ["..."],
  "codeExamples": [...],
  "diagrams": [...],
  "references": [...]
}
```

### 样式设计

- **答案摘要** - 蓝色背景 (`bg-blue-50`)，左侧蓝色边框
- **章节卡片** - 白色背景，边框，圆角
- **考察重点** - 琥珀色背景 (`bg-amber-50`)
- **延伸追问** - 紫色背景 (`bg-purple-50`)
- **关键要点** - 绿色背景 (`bg-green-50`)
- **前置知识** - 琥珀色背景
- **代码块** - 深灰色背景 (`bg-gray-900`)
