# Mermaid 图表安全配色指南

> 最后更新：2026-04-23
> 
> 所有配色经过 WCAG AA 对比度验证（≥ 4.5:1），深色背景 + 白色文字。

## 📐 配色原则

1. **深色背景 + 白色文字** — 唯一安全组合
2. **使用 Tailwind 800-900 色阶** — 确保足够深色
3. **同一色系内保持一致性** — 红色系用红色，绿色系用绿色
4. **禁止使用 400-500 色阶** — 太浅，配白字对比度不足

## 📝 style 语法模板

```mermaid
style NodeName fill:#深色,stroke:同色系边框,color:#fff
```

**完整示例：**
```
style RootNode fill:#991b1b,stroke:#dc2626,color:#fff
style ProcessNode fill:#1e3a5f,stroke:#60a5fa,color:#fff
style SuccessNode fill:#064e3b,stroke:#059669,color:#fff
style WarningNode fill:#92400e,stroke:#d97706,color:#fff
```

## 🧪 验证方法

提交前运行：
```bash
node scripts/validate-article.mjs src/data/articles/你的文章.ts
```

如果配色不达标，脚本会报错并给出替代建议。
