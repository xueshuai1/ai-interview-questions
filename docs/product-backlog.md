## 💡 给开发的具体建议

### 本轮开发优先级
1. **先更新首页统计数字**：修改首页组件中的硬编码数字，与实际数据同步
2. **添加工具**：在 tools.ts 中添加 8 个新工具，注意分类和标签
3. **更新 GitHub Stars**：同步 trending 项目的 star 数
4. **构建测试**：`npm run build` → `npx tsc --noEmit` → 确认编译通过
5. **部署**：运行 `./scripts/deploy.sh` 更新首页时间

### 代码改动注意事项
- 确保 `npx tsc --noEmit` 编译通过再推送
- 每次改动后运行 `npm run build`
- 首页时间必须更新
- 工具添加时注意分类映射
